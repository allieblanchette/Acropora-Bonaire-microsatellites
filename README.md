# Acropora-Bonaire-microsatellites
Microsatellite allele tables and data analysis R script from Caribbean Acropora spp. genotyping study conducted with Reef Renewal Foundation Bonaire.

Blanchette_et_al_2026_data_analyses.R: R code script for all final analyses conducted for this project
Allele tables: folder of csv files of allele tables for Acropora cervicornis and Acropora palmata, diploid formatted. There is also a triploid-formatted csv for A. cervicornis, which had several instances of triploidy
Other data for R: folder of csv files of other various data (e.g., metadata, literature comparison data, data formatted specifically for iNext) used in analysis code
